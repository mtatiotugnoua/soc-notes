# Commandes Git Essentielles

## Installation & Config
- git --version
- git config --global user.name "Nom"
- git config --global user.email "email"

## Travailler avec un repo
- git clone [url]
- git status
- git add .
- git commit -m "message"
- git push
- git pull

## Branches
- git branch
- git checkout -b nom-branche
- git merge nom-branche
